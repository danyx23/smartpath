Summary

Represents an absolute filesystem directory path. Valid paths are windows NTFS paths starting with a drive letter (C:\test) or SMB Shares (\Server\share\somedir). Whenever an AbsoluteDirectory is converted back into a string, a backslash character will be appended to distinguish it from filenames
